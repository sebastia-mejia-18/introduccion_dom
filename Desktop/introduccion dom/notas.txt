metodos de los arreglos: estos metodos son solo para arreglo (osea que esten dentro de un arreglo)

  map()-> es un metodo de iteracion que manipula todo tipo de arreglo, el cual retornara un nuevo
  tipo de arreglo

  filter()-> es un metodo de iteracion que filtra el contenido 
  de un arreglo bajo una condicion retornado un nuevo arreglo

  find()-> es una iteracion que busca un unico elemento dentro den un arreglo
  y retorna ese elemento

  some()-> iteracionque busca un unico elemento dentro de un arreglo y retorna true o false

  push()-> es un metodo que agrega un metodo que agrega un elemento a la ultima posicion 
  del arreglo actual

  pop()-> metodo que elimina la ultima posicion de un arreglo

  shift()-> metodo que elimiina la primera posicion de un arreglo

  unshift()-> metodo que agrega un elemento a la primera posicion de un arreglo

  splice()-> metodo que elimina un metodo de una posicion 'n' de un arreglo
  (n puede ser cual quier valor)

  forEach()-> lo mismo que el map pero sin retorno de arreglo


  Metodos Mutables-> modifica el arreglo original

  Metodos no Mutables-> NO modifica el arreglo original
